Aqua Pomus Fruit Infuser Bottle

This group project was developed for both a 3D Animation and an Electronic Image Production class at Fanshawe College.

Objective

The main goal was to create the brand identity, develop a unique bottle design and build it on Cinema 4D. The only specification was that it needed to have an infuser. We decided to create a fruit infuser bottle.

Result

We decided on the name Aqua Pomus, meaning water fruit in Latin. The logo and the identity are clean and simple, but very colorful, with the objective of making the brand more approachable and playful. The main feature that would differentiate the bottle is it's unique lid.
